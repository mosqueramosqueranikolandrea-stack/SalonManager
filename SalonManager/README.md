# SalonManager

Sistema de gestión para un salón de belleza desarrollado en C# utilizando una aplicación de consola.

## Funcionalidades

- Registrar clientes
- Registrar servicios
- Registrar citas
- Mostrar información almacenada

## Tecnologías utilizadas

- C#
- .NET
- Git
- GitHub

## Estructura del proyecto

SalonManager
│
├── Modelos
├── Servicios
└── Program.cs

## Descripción

Este sistema permite administrar la información básica de un salón de belleza.  
El usuario interactúa con el sistema mediante un menú en consola para gestionar clientes, servicios y citas.
